Эта функция позваляет преобразовывать обычную функцию, должно передаваться определенное количество параметров в каррирующую функцию.
Например, есть функция:
function sum(a, b, c, d) {
    return: a + b + c + d;
}
Чтоы она отработала так как нужно в неё необходимо передать 4-е параметра.
Преобразовав эту функцию в каррирующую:
var currySum = carry(sum)
появляется возможность передать в нее параметры в несколько шагов:
currySum(2)(3, 4)(5);
